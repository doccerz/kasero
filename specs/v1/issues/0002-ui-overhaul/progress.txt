## Notes / Learnings

- Next.js 16.2.1: `middleware.ts` is DEPRECATED — must use `proxy.ts` with `export function proxy()`
- Route groups (protected) in Next.js app router: old pages at same URL path must be deleted to avoid
  "parallel pages that resolve to the same path" build error
- Playwright `page.route()` only intercepts BROWSER-side fetches, not server-side Node.js fetch calls
  → Admin pages (server-side fetch) require a separate mock HTTP server (globalSetup) on a dedicated port
  → Entry form POST (client-side fetch) can use page.route() directly in the test
- Stale .next/ cache causes false tsc errors after moving pages — always `rm -rf .next` before type-checking
  after restructuring routes
